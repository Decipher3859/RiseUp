## UX
- [ ] Die App vermittelt keine Anspruchshaltung an den User
    - [ ] keinen Druck aufbauen
    - [ ] keinen Vergleich aufdrängen
    - [ ] keine Leistungserwartungen
- [ ] Statistiken für Reflexion anbieten
- [ ] Unmittelbares, visuelles Feedback
    - [ ] erledigte Gewohnheit = User "gießt" Pflanze => Pflanze wächst
    - [ ]  Gewohnheit ausgelassen => Pflanze geht einen Schritt zurück (sie geht nicht ein, die Animation geht nur einen Schritt zurück)
